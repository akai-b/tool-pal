# ToolPal 图片处理工具 — 产品需求文档 v1.0

## 产品定位

面向前端开发者的本地图片处理桌面工具，核心价值：
- **本地处理**：所有操作在本地完成，不依赖云服务，无隐私顾虑
- **快捷操作**：集成系统右键菜单，无需打开 GUI 即可完成常用操作
- **批量处理**：支持文件夹级批量操作，保留目录结构

---

## 功能模块

### 一、图片压缩

- 支持输入格式：JPEG、PNG、WebP
- 支持输出格式：JPEG、PNG、WebP（可与输入格式不同）
- 质量调节（1–100）
- 尺寸限制：可设置最大宽度 / 最大高度，等比缩放
- 压缩前后体积对比展示（原始大小、压缩后大小、节省百分比）
- 批量压缩：支持同时拖入多张图片或整个文件夹
- 输出方式（可配置）：
  - 输出到原文件同目录，文件名加后缀（如 `image_min.jpg`）
  - 覆盖原文件
  - 输出到指定目录

---

### 二、格式转换

- 支持输入格式：JPEG、PNG、WebP、AVIF、GIF
- 支持输出格式：JPEG、PNG、WebP、AVIF
- 重点场景：PNG ↔ WebP 互转
- 批量转换：支持多文件 / 整个文件夹
- 转换后保留原始文件名，仅修改扩展名
- 批量转换文件夹时，可选是否保留子目录结构
- 质量可配置（有损格式）；PNG 等无损格式不显示质量选项

---

### 三、动图制作

#### 3.1 序列帧 → 动图合成

- 输入：多张静态图片（JPEG / PNG / WebP），支持拖拽上传
- 帧管理：
  - 缩略图列表展示所有帧
  - 支持拖拽调整帧顺序
  - 支持单帧删除
- 可配置参数：
  - 帧率（1–60 fps）
  - 循环次数（指定次数 / 无限循环）
  - 输出尺寸（宽 × 高，等比缩放）
- 输出格式：GIF、WebP 动图

#### 3.2 动图编辑

- 输入：已有 GIF 文件
- 编辑功能：
  - **裁剪**：可视化框选裁剪区域，所有帧同步裁剪
  - **缩放**：指定宽/高或百分比，等比缩放
  - **帧率调整**：整体加速 / 减速
  - **帧管理**：查看所有帧缩略图，支持删除指定帧、调整帧顺序
  - **帧提取**：将 GIF 拆分导出为序列帧图片（JPEG / PNG）
- 实时预览：编辑参数时可预览动图效果

#### 3.3 视频片段 → GIF（延伸功能）

- 输入：MP4 / WebM 短视频（建议限制 ≤ 30s）
- 可配置：截取起止时间、输出帧率、输出尺寸
- 输出格式：GIF

---

### 四、系统右键菜单集成

> 用户在文件管理器中右键图片文件或文件夹时，可直接触发 ToolPal 的常用操作，无需手动打开应用。

#### 4.1 右键单张图片文件

显示子菜单「ToolPal」，包含：

| 菜单项 | 行为 |
|--------|------|
| 压缩图片 | 使用默认配置快速压缩，完成后显示系统通知 |
| 转换为 WebP | 转换格式，完成后显示系统通知 |
| 转换为 PNG | 转换格式，完成后显示系统通知 |
| 转换为 JPEG | 转换格式，完成后显示系统通知 |
| 在 ToolPal 中打开 | 打开 GUI 并加载该文件 |

#### 4.2 右键文件夹

显示子菜单「ToolPal」，包含：

| 菜单项 | 行为 |
|--------|------|
| 批量压缩文件夹内图片 | 递归处理文件夹内所有图片，完成后显示系统通知 |
| 批量转换为 WebP | 批量转换，完成后显示系统通知 |
| 批量转换为 PNG | 批量转换，完成后显示系统通知 |
| 在 ToolPal 中打开文件夹 | 打开 GUI 并加载该文件夹 |

#### 4.3 右键操作的通用行为

- 操作完成后显示系统通知，包含：成功数量、失败数量（如有）
- 快速操作使用预设的默认配置（可在设置中修改）
- 输出文件默认放在原文件同目录，命名规则可在设置中配置

---

### 五、设置中心

- **默认压缩质量**：右键快速压缩时使用的质量值
- **默认输出格式**：右键快速转换时的目标格式
- **输出命名规则**：后缀模式（`_min`、`_webp` 等）/ 覆盖原文件 / 输出到指定目录
- **批量处理范围**：是否递归处理子文件夹
- **主题**：跟随系统 / 亮色 / 暗色
- **语言**：中文 / English

## 界面结构

```
ToolPal
├── 图片压缩
├── 格式转换
├── 动图制作
│   ├── 序列帧合成
│   ├── GIF 编辑
│   └── 视频转 GIF
└── 设置
```

---

## 优先级

| 优先级 | 功能模块 |
|--------|----------|
| P0 | 图片压缩、格式转换（PNG ↔ WebP 为核心场景） |
| P1 | 系统右键菜单集成（文件 + 文件夹） |
| P2 | 动图制作（序列帧合成 + GIF 编辑） |
| P3 | 视频转 GIF、设置中心、主题/多语言 |
